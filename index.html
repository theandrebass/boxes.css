boxes.css adds a layer of dimension to your web elements. It's light enough that it takes no time to set up or load, but interesting enough that it can serve as the only major CSS on the page.

Basics

boxes.css is a single stylesheet. Get started by adding the stylesheet to your page's head.

<link rel="stylesheet" href="https://unpkg.com/boxes.css/dist/boxes.min.css" />

You can make any element boxy by giving it the class box.

<button class="box">Hello, World!</button>

Wrapper boxes

Sometimes, you want to boxify an element like img or input. But these elements can't have children like ::before and ::after elements, so boxes.css can't work its magic. Instead you'll have to wrap them in a wrapper box, using the wrapper class, like this. (The wrapper class removes padding in the box.)

Inline boxes

Theming

You can theme button styles by customizing the colors, using CSS custom properties.

There are four variables that boxes.css uses:

  • --box-text-color determines the color of text, and the border around the box and its shadow

  • --box-background-color is the color of the button background

  • --box-accent-color is the background color of accented buttons

  • --box-shadow-color is the fill color of the shadow
